# MSCEIS 2019 Vue

[MSCEIS 2019](https://2019.msceis.org/)

* Vue 2.6.6
* Vue-router 3.0.1
* Vuetify 1.3.0

## Project setup
```
yarn install
```
_or_
```
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
```
_or_
```
npm run serve
```

### Compiles and minifies for production
```
yarn build
```
_or_
```
npm run build
```

### Linting and Fixes
```
yarn run lint
```
_or_
```
npm run lint
```
